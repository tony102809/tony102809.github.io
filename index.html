Resume / Past Projects

Below are past projects I have done in many different langauges:

NYU High Performance Computing Team (HPC)

Member on the NYU HPC team, specifically the HPC Conjugate Gradient (HPCG) software group. The HPCG benchmark uses a preconditioned conjugate gradient (PCG) algorithm to measure the performance of HPC platforms with respect to frequently observed but challenging patterns of computing, communication, and memory access.

Mountain Climb (Java)

Implemented a console-based game using a binary search tree to model a mountain hiking expedition, allowing the hiker to find safe paths with limited resources and overcome obstacles. Developed a customized BST class and nested Node class to represent the mountain's rest-stops, considering supplies and obstacles at each point. Designed an efficient iterator interface for in-order, pre-order, and post-order traversals, enabling seamless access to the paths and ensuring optimal program performance.

Translation Lookaside Buffer Translation (C Language)

Developed and implemented an automated Translation Lookaside Buffer (TLB) simulation program in C, capable of retrieving byte data based on virtual addresses. Developed functionality to read and store information about TLB contents, virtual page table, and physical memory cache. Created a user interface to prompt for virtual addresses, retrieving the corresponding byte data or displaying “Cannot be determined” if the data is unavailable. Ensured adherence to coding conventions, proper documentation, and memory management practices.

Decoding Sequences With MDeques (Java)

Developed an advanced program using MDeque, a customized double-ended queue with fast access to its middle element, to decode numerical sequences efficiently. Implemented a recursive algorithm and optimized the implementation to meet performance requirements, ensuring O(1) time complexity for push, pop, and peek operations. Completed the implementation of isValid and decode functions, utilizing a doubly linked list design for the MDeque data structure and enabling seamless decoding based on user-provided decoding keys.

Trees on NYC Streets (Java)

Developed a console-based program leveraging open data from the NYC Department of Parks, parsing and analyzing a large tree dataset from the 2015 Street Tree Census. Utilized advanced programming techniques, including multi-file architecture, command line argument handling, and efficient data structures like lists and classes. Implemented a user-friendly interface for searching and retrieving tree species information, showcasing expertise in data processing, program design, and software development best practices.

Reverse Engineering (C / Assembly Language)

Implemented C code for multiple designated assembly functions, achieving functional parity with the provided version and passing rigorous testing.
